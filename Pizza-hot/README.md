## ğŸš€ Overview

**Pizza Hot** is an interactive React-based pizza ordering application built with a modern and responsive design, demonstrating best practices in frontend development. This project is ideal for anyone learning React, Context API, custom hooks, state management with reducers, and working with backend APIs.

---

## ğŸ¯ Project Goals

This project is created for educational purposes to demonstrate:

- **React Context API** for state management (cart, theme, and UI state).
- Custom hooks (`useFetch`) to manage API requests effectively.
- Advanced state management using **Reducers** (`useReducer`).
- Responsive and interactive UI using **Bootstrap** and custom CSS.
- Building and consuming a backend API using **Express**.

---

## âœ¨ Key Features

- **Dynamic Pizza List:** Fetch pizza details from backend API.
- **Interactive Cart:** Add, remove, increase/decrease quantity of pizzas.
- **Checkout Modal:** Complete checkout form with client-side validation.
- **Theme Selector:** Choose your favorite color theme and toggle between light/dark mode.
- **Seamless Backend Integration:** Node.js backend built with Express for data storage and order processing.

---

## ğŸ› ï¸ Technologies Used

- **Frontend:**
  - React.js (Hooks, Context API, Reducers)
  - Bootstrap 5
  - Custom CSS & Responsive Design
- **Backend:**
  - Node.js with Express.js
  - JSON data storage
  - CORS management
- **Others:**
  - Vite
  - React Portals for Modals
